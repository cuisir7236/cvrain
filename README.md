vrain - Typst 仿古籍直排电子书模板

vrain 是一款专为 Typst 打造的模板，用于快速生成仿古籍直排（竖排）格式的电子书（PDF），核心支持楷体正文、宋体双行夹注、竖排标点自动转换、自定义符号替换、标点位置微调、批注间距灵活调整及页面背景图添加等功能，适配古籍排版的视觉与格式需求。

快速使用步骤

+ 将模板核心文件 vrain.typ 放置在你的 Typst 项目根目录中；
+ 在项目主文档中导入模板：#import "vrain.typ": vrain；
+ 准备正文内容，使用【】 包裹需要设置为双行夹注的批注文字；
+ 调用 vrain 函数并根据需求传入对应参数，执行编译即可生成直排电子书。

固定替换：如 "@": " "，将普通 @符号直接替换为单个空格；
数字空格：正文内输入@数字（如@5），将自动替换为对应数量的连续空格。

#### 正文内容标记规则
+ 普通文字：直接输入字符，模板将按直排规则正常渲染；
+ 双行夹注批注：使用【】 包裹批注内容，示例：天地玄黄【天为阳，地为阴】宇宙洪荒，批注将自动以双行夹注形式呈现（右列自上而下、左列自上而下）；
+ 指定数量空格：使用@数字格式，示例：@3生成 3 个连续空格、@8生成 8 个连续空格，适配古籍排版的空格需求；
+ 自定义符号替换：通过at-replace-map参数配置键值对，实现任意符号的自定义替换，适配个性化排版需求。
字体配置说明
+ 模板内部默认硬编码了示例字体名称（楷体、宋体均为 "AaHLGGT"），该字体为测试示例，需根据自身系统已安装的字体修改；
+ 修改位置：打开模板文件vrain.typ，找到vrain函数开头附近的text-fonts（正文字体配置）和comment-fonts（批注字体配置）变量，替换为目标字体名称；

### 注意事项
+ 模板仅支持 Typst 编译环境，需确保本地已安装 Typst 或使用在线 Typst 编辑器；
+ 背景图建议使用与页面尺寸比例适配的图片，减少裁剪或留边，提升视觉效果；
+ 调整列数、行数、字号时，建议整体适配，避免出现文字溢出、排版错乱的情况；
+ 若开启竖排标点转换后出现标点显示异常，可关闭该功能（设为 0），手动使用竖排专用标点。
